# Proposal Presentation Slides

## Submission Instructions

Submit your proposal presentation slides via pull request to the `project-proposal` branch in this directory by **11:59 AM on May 5, 2026**.

## Presentation Format

- **Duration**: 5-7 minutes
- **Maximum Slides**: 6 slides total
  - 1 title slide
  - 5 content slides

## Required Content

Your proposal presentation should include:

1. **Title Slide**
   - Project title
   - Your name
   - Date

2. **Project Overview & Motivation**
   - What problem are you solving?
   - Why is this interesting or important?

3. **Data Collection Approach**
   - Data source(s)
   - Web scraping method or API usage
   - Current progress on data collection

4. **Exploratory Data Analysis**
   - Initial data visualizations
   - Key patterns or insights discovered
   - Data quality and preprocessing needs

5. **Proposed Modeling Approach**
   - Type of model(s) you plan to use
   - Features you plan to engineer
   - Evaluation metrics

6. **Timeline & Next Steps**
   - Key milestones and timeline to completion
   - Potential challenges and mitigation strategies
   - Current status and remaining work

## File Naming Convention

Please name your file: `lastname-firstname-proposal.pdf` or `lastname-firstname-proposal.pptx`

Example: `smith-john-proposal.pdf`

## Presentation Order

Presentation order will be randomized and announced before the presentations begin.
